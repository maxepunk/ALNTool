Force-Directed Layout Test

Open the Journey Intelligence View at http://localhost:3000/journey-intelligence

Expected Behavior:

  • When the graph loads, nodes should spread out automatically
  • Characters should be well-separated (>150px apart)
  • Elements owned by characters should cluster near their owners
  • The simulation should run for a few seconds then stabilize

To Test:

  1. Open the Journey Intelligence View
  2. Wait for nodes to load
  3. Watch the nodes spread out with force-directed physics
  4. Open browser console and paste the test script from test-force-layout.js
  5. Check the console output for node movement and clustering metrics

Visual Indicators:

  • Characters (circles) should be far apart
  • Elements (diamonds) should group near their owner characters
  • The layout should look organic, not grid-like
  • No nodes should overlap

Debugging:

If nodes aren't moving, check:

  • Browser console for errors
  • Network tab to ensure data is loaded
  • React DevTools to check if useGraphLayout hook is active
